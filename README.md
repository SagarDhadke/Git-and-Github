## All Commands

Make directory / Folder 

```bash
  mkdir ProjectName
```

Show all files in desktop

```bash
  ls
```

Chnage directory

```bash
  cd fileName
```

Get Hidden Files like git and etc...

```bash
  git init

  //-a means show me all the hidden files
  
  ls -a
```
